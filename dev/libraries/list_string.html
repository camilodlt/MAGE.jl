UTCGP.liststring_caps.capitalize_list_string — Function
capitalize_list_string(strings::Vector{String}, args...)::Vector{String}

Broadcasts the Base.titlecase function to every element in the vector.

source
julia> capitalize_list_string(["julia", "julia"])
2-element Vector{String}:
 "Julia"
 "Julia"
julia> capitalize_list_string(["julia julia", "julia"])

 "Julia Julia"
 "Julia"

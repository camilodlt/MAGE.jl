UTCGP.number_arithmetic.safe_div — Function
safe_div(a::Number, b::Number, args...)

If the divisor is equal (==) to 0, then the function returns 0 and does not raise an Error.

Else, the division works as expected.

source
julia> safe_div(3,0)
0
